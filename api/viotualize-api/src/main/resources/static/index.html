Swagger Documentation What do you expect?

Path Description
/metrics Shows (Spring Boot built-in) metrics information for the current application.
/codahale-metrics Shows Codahale Metrics (API metrics, defined via @Timed and @ExceptionMetered).

Path Description
/autoconfig Displays an auto-configuration report showing all auto-configuration candidates and the reason why they "were" or "were not" applied.
/beans Displays a complete list of all the Spring Beans in your application. Very cool
/configprops Displays a collated list of all @ConfigurationProperties
/env Exposes properties from Spring's ConfigurableEnvironment.

Path Description
/trace Displays trace information (by default the last few HTTP requests).

Path Description
/health Shows application health information (defaulting to a simple "UP" message).

Path Description
/info Arbitrary application info (e.g., Maven & Git info)

Path Description
/dump Performs a thread dump.
/shutdown Allows the application to be gracefully shutdown (not enabled by default).
