Features

Find and Integrate the Right Devices
Search and find suitable "things" and devices in a comprehensive catalog. Manufacturers are able to maintain detailed specifications of their devices.
Define and Provision Virtual Sensors
No need to purchase equipment or set up expensive lab environments or field tests to develop and verify your application logic. Choose from hundreds of virtualized sensors or create your own.
Easily Spin up Realistic Environments
Sensors donâ€™t live in isolation. Create realistic scenarios composed from multiple sensors and actuators. Model the world as you need it for your application.
Robustness Testing via Simulations.
Things can go wrong. And they probably will. Using our robustness testing module, you can make sure your application can gracefully handle sensor failures, quality degradations, or spurious data. Additionally, data and interaction traces are captured from your infrastructure and can be played back to you for testing.
Test Using the Production Protocol
No need to adapt your application for testing. Access your virtualized IoT environment using the actual sensor protocols. Maximum dev/production parity!
